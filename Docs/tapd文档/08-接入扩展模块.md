# 接入扩展模块

## 配置扩展模块

应用开发者团队可以在开发者后台配置应用需要的扩展模块（开发者后台 -> 选择应用 -> 应用开发 -> 入口配置）

输入json格式的扩展模块配置，点击保存，即可配置完成。

常见配置信息：
1. 显示名称 - name
2. 入口地址 - url

每一个扩展模块均可实现TAPD扩展UI定制

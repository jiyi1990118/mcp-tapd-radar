# 应用权限控制

## 配置应用权限

应用开发者团队可以在开发者后台配置**应用权限**（开发者后台 -> 选择应用 -> 应用开发 -> 应用权限)

![应用权限](/document/img/应用权限.png)

## 使用应用权限

### API权限控制

开放应用只能使用开放应用已拥有的**应用权限**的对应**模块**的API。

### Webhook权限控制

TAPD只会成功发送开放平台已拥有的**应用权限**的对应**模块**的webhook。

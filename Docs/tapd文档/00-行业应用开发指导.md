# TAPD 行业应用开发指导

## 创建应用

应用开发者团队首先需要在TAPD开放平台创建应用，你可以在这里：

* 定义应用名称、唯一ID
* 设置应用类别、是否托管
* 上传应用图标、应用介绍

## 开发应用

现在你已经创建好属于自己的应用了，接下来进入开发应用环节。应用开发者团队也可以在这里对应用进行调试工作

* 配置挂载点
* 配置应用权限
* 测试应用

主要内容：

* 接入扩展模块
* 应用权限控制
* 使用API
* 使用Webhook
* 安全IP控制
* TAPD核心概念
* TAPD API文档
* 扩展模块介绍

## 发布应用

应用开发完成后，可以进入应用发布审批环节，审批通过后会自动发布最新审批通过的应用版本

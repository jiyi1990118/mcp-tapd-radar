# 快速接入

Tapd 开放平台支持公司内各个业务团队定制开发，可以基于界面接入点入口或者基于Tapd API数据能力进行系统集成。

## 接入步骤

## 创建应用

应用开发者团队需要在TAPD开放平台创建应用(开发者后台 -> 创建应用)

![创建应用](/document/img/cloud/创建应用.png)

## 接入点配置

Tapd基于系统基础能力，在核心页面入口提供嵌入能力。具体可以参考接入点列表

默认推荐项目导航应用下的接入点，该接入点拥有着主导航栏展示功能，示例如下：

![项目导航示例](/document/img/cloud/项目导航示例.png)

需要配置如下接入点信息：（开发者后台 -> 选择应用 -> 应用开发 -> 入口配置）

1. 应用名称 - name
2. 主页入口地址 - url

![项目导航挂载点配置](/document/img/cloud/项目导航挂载点配置.png)

配置成功后需要进行测试部署，在测试项目中查看效果。

## 应用发布

应用开发完成后，可以发布应用上架应用商店（开发者后台 -> 选择应用 -> 应用发布）

![选择部署版本](/document/img/选择部署版本.png)

# 安全IP控制

应用开发者团队可以在开发者后台配置**安全IP**，来控制API访问安全（开发者后台 -> 选择应用 -> 应用开发 -> 安全设置)

如果需要输入IP段，使用通配符*, 如116.31.81.*。

![安全设置](/document/img/安全设置.png)

## 使用安全IP请求API

如果开放应用设置了安全IP控制，开放应用只能使用开放应用已设置的**安全IP**去请求API。
